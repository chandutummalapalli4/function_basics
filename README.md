# 🧠 Key Concepts Learned

## Function

A function is a reusable block of code that performs a specific task.

---

## Parameter

A parameter is a variable declared in the function definition that receives values.

Example:

```python
def greet(name):
```

---

## Argument

An argument is the actual value passed to a function when it is called.

Example:

```python
greet("Chandu")
```

---

## Positional Arguments

Python assigns arguments to parameters based on their position.

Example:

```python
student("Chandu", 21)
```

Internally:

```python
name = "Chandu"
age = 21
```

---

## `print()` vs `return`

### `print()`

* Displays output on the screen.
* Does not return a value.
* Functions without a `return` statement automatically return `None`.

### `return`

* Sends a value back to the caller.
* Allows the returned value to be stored and reused.
